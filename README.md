# 机器学习 学习笔记
## Avik-Jain machine learning 100day 项目学习笔记
### day1
学习如何导入数据、处理丢失的数据以及分解训练集&验证集

1.需要导入numpy和pandas库

2.loc和iloc都是摘取数据矩阵中数据的函数，loc是按照行和列的命名摘取的，而iloc则是根据行号和列好摘取的，举个例子：

对于一个下面的矩阵
    A   B   C   D
a   0   1   2   3
b   4   5   6   7
c   8   9  10  11
d  12  13  14  15

loc['a']取的数据为
table1|A|B|C|D
---|---|---|---|---   
a|0|1|2|3

而iloc[0]取的数据同样为
table1|A|B|C|D
---|---|---|---|---   
a|0|1|2|3
这是由于iloc是从0开始定义行or列的
